# Analytical Geometry Cheatsheet

This directory contains the English version of the cheatsheet, designed in LaTeX. It serves as an ideal reference material for students, engineers, and researchers in mathematics, physics, and computer graphics. The document concisely summarizes essential formulas and concepts for both 2D and 3D coordinate geometry.

---

## Core Content

* **2D Cartesian Plane:** Distance, midpoint, division by ratio, shoelace formula, and all forms of the straight line equation (point-slope, general, symmetric).
* **Conic Sections & Polar Coordinates:** Detailed properties (discriminant, foci, directrices, latus rectum) for the circle, parabola, ellipse, and hyperbola, along with coordinate conversions and the universal polar equation.
* **Vectors ($\mathbb{R}^2, \mathbb{R}^3$):** Dot/Cross products, vector projections, and the scalar triple product.
* **3D Analytical Geometry:** Equations for lines and planes in space, distances between skew lines and parallel planes, and standard equations for quadric surfaces (Ellipsoid, Paraboloids, Hyperboloids, Cone).
* **Matrices & Linear Transformations:** Geometric transformations (rotation, scaling) and the use of homogeneous coordinates for translations via affine transformations.

---

## Usage and Compilation

If you wish to build the PDF document directly from the source code (`analytical_geometry_cheatsheet.tex`), you can use your preferred LaTeX environment or run the following command in your terminal using `pdflatex`:
```bash
pdflatex analytical_geometry_cheatsheet.tex
```

*Note: Depending on your setup, you may need to compile the document twice to ensure all cross-references and formatting are generated correctly.*
