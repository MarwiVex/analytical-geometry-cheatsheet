# Formulario de Geometría Analítica

Este directorio contiene la versión en español del formulario, diseñado en LaTeX. Sirve como un material de referencia ideal para estudiantes, ingenieros e investigadores en matemáticas, física y gráficos por computadora. El documento resume de manera concisa las fórmulas y conceptos esenciales para la geometría de coordenadas en 2D y 3D.

---

## Contenido Principal

* **Plano Cartesiano 2D:** Distancia, punto medio, división por razón, fórmula del área de Gauss (Shoelace), y todas las formas de la ecuación de la línea recta (punto-pendiente, general, simétrica).
* **Secciones Cónicas y Coordenadas Polares:** Propiedades detalladas (discriminante, focos, directrices, lado recto) para la circunferencia, parábola, elipse e hipérbola, junto con conversiones de coordenadas y la ecuación polar universal.
* **Vectores ($\mathbb{R}^2, \mathbb{R}^3$):** Productos punto/cruz, proyecciones vectoriales y el triple producto escalar.
* **Geometría Analítica 3D:** Ecuaciones para rectas y planos en el espacio, distancias entre rectas cruzadas y planos paralelos, y ecuaciones estándar para superficies cuádricas (Elipsoide, Paraboloides, Hiperboloides, Cono).
* **Matrices y Transformaciones Lineales:** Transformaciones geométricas (rotación, escalado) y el uso de coordenadas homogéneas para traslaciones mediante transformaciones afines.

## Uso y Compilación

Si deseas compilar el documento PDF directamente desde el código fuente (`geometria_analitica_cheatsheet.tex`), puedes utilizar tu entorno LaTeX preferido o ejecutar el siguiente comando en tu terminal usando `pdflatex`:
```bash
pdflatex geometria_analitica_cheatsheet.tex
```

*Nota: Dependiendo de tu configuración, es posible que necesites compilar el documento dos veces para asegurar que todas las referencias cruzadas y el formato se generen correctamente.*
